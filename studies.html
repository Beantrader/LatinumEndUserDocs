Studies¶

Latinum includes over 200 indicators [provided by TA-Lib] such as ADX, MACD, RSI, Stochastic, Bollinger Bands, and also includes candlestick pattern recognition. In addition to the classic indicators provided by TA-Lib, Latinum also includes Daily Pivot Point, Volume by Price, and studies that color candlesticks or display arrows based on moving average crossover.

In Latinum indicators are located under the Studies menu bar item.

Inserting¶

To insert an indicator:

  1. Select Studies > Insert Studies from the menu bar. A dialog window displaying the available indicators is displayed.
  2. You can browse available indicators or use the text entry box near the top of the dialog window to filter the indicators by name (e.g. RSI) or description (e.g. Relative Strength Index).
  3. When you find the indicator you want select it using the arrow keys (or mouse), and press Enter (or select the Apply button). Another dialog window will be displayed which allows you to specify indicator specific parameters, and colors.
  4. Select the OK button to attach the indicator to the active chart.

Editing¶

To edit the parameters of an indicator already attached to a chart:

  1. Select Window > Outliner. A tool window is displayed showing a tree representation of a charts indicators (or Studies) and drawings.
  2. Right-click the indicator in and select Edit Attributes.

Removing¶

To remove an indicator from a chart:

  1. Select Window > Outliner.
  2. Right-click the indicator in and select Delete
