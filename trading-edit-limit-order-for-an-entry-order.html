Edit Limit Order for an Entry OrderÂ¶

  1. In the Orders table, right-click an entry order, and then select Edit Limit Order.

  2. The Edit Limit Order dialog box will be shown.

  3. Select Rate to specify the Limit (Take Profit) order price, or select Pips to specify the Limit order in pips offset from the orders entry price.

    If you enter the value in pips, the value is positive for buy entry orders, negative for sell entry orders.

    If there is no existing Limit order attached to the Entry order you put a tick into the Limit checkbox. To remove the Limit order uncheck the Limit checkbox.

  4. Select OK to submit your modified order, or Cancel to close the dialog discarding your changes.

    If no errors occur the dialog box will close automatically, otherwise the error message will be displayed.
