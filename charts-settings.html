Charts Settings¶

Changing the Symbol¶

Main Menu Method:¶

  1. With the chart sub-window active, select Charts > Symbol Search menu item
  2. Select USD/JPY or some other symbol
  3. Select OK

Chart Toolbar Method:¶

  1. Right-Click (or Control+Click on Mac) the chart sub-window, ensure Show Toolbar is checked in the context menu
  2. The chart will have a toolbar allowing you to change the symbol using a combo-box

Changing the Interval¶

Main Menu Method:¶

  1. Activate the chart you want to change the interval of by clicking on its title bar or a blank space within the chart
  2. Select an Interval from the Interval main menu

Context Menu Method:¶

  1. Right-click the chart
  2. Select the Interval sub-menu
  3. then select the Interval

Chart Toolbar Method:¶

  1. Show the chart toolbar by right-clicking (or Control+Click on Mac) the chart
  2. Ensure Show Toolbar is checked
  3. Change the Interval selected on the chart toolbar

Chart Scaling¶

Plot Top and Bottom Margin¶

  1. Select Chart > Chart Properties main menu entry
  2. From the Chart Properties dialog, select the Scaling tab
  3. Ensure Use Margins is checked
  4. Enter a percentage of the charts height to use for the top and bottom margins
  5. Select OK

Change Scale Type¶

You can select from Linear, or Semi-Log chart scaling.

  1. Right-click the charts price axis and select Linear Scale for Linear scales, or select Log Scale for Semi-Log chart scales
